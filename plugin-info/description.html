Plugin support for GLSL.

Features:

  • Identifiers References
  • Code completion
  • Refactoring
  • Formatter
  • Syntax highlighting
  • Go to / find usage
  • Documentation for GLSL builtin functions
  • Support inside HTML
  • Macro support

Supported file extensions:

  • .glsl
  • .vert
  • .vsh
  • .tesc
  • .tese
  • .geom
  • .gsh
  • .frag
  • .fsh
  • .comp

Absolute path resolution for #include:

  • With a marker file (.../whatever/.glsl_idea_root)
  • MC-style "shaderpack" folders (.../shaderpacks/whatever/...)
The marker file takes precendence.
